Como usar as tags Heading

(Voltar)

As tags Heading são tags para destacar títulos e subtítulos para a sua pagina. De H1 até a H6, sendo H1 tendo a propriedade de maior tamanho e H6 de menor tamanho. De modo geral, a H1 deve ser utilizada como o título das páginas e é um elemento único em cada página. Depois disso, temos o H2, que está hierarquicamente submetido ao H1, e o H3 que por sua vez está submetido ao H2 e assim sucessivamente. Não utilize para alterar tamanho de fonte. A melhor forma de fazer isso é no CSS.

Como Fazer uma citação em HTML

(Voltar)

Para fazer uma citação no documento HTML se usa o elemento blockquote que define uma seção dentro de um documento que é citada de outra fonte. Por exemplo:

< blockquote >"A força não provém da capacidade física. Provém de uma vontade indomável." Gandhi< blockquote>

E fica assim:

"A força não provém da capacidade física. Provém de uma vontade indomável."Gandhi

Também pode ser colocado uma URL para que a citação possa vir direto da fonte, caso seja de algum site de estudos. Para alterar o recuo e o tamanho, a melhor opção seja o CSS.

Listas ordenadas e não ordenadas

(Voltar)

As listas ordenadas começam com a tag < ol > e coloca todos os itens em ordem numérica. O navegador entende que os itens listados são colocados em ordem crescente de importância. Sewgue o exemplo:

  1. item branco
  2. item roxo
  3. item vermelho
  4. item preto

Já as listas não ordenadas, são colocadas em ordem com marcadores apenas, sem grau de importancia ou ordem crescente. A tag usada para essa lista é < ul >, veja o exemplo:

  • item branco
  • item roxo
  • item vermelho
  • item preto

listas são usadas também para fazer menu em front-end. Além disso, para usar as listas corretamente, tanto a tag ol ou ul precisam ter a tag filha li. Ela que lista os itens dentro das tags pai.
